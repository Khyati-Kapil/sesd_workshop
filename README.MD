Expense Tracker Backend

A REST API built using Node.js, Express, TypeScript, and MongoDB to manage expenses.
The project follows Object-Oriented Programming (OOP) principles with a clean architecture using controllers, services, and repositories.


Features

Create, read, and delete expenses

Search by title

Sorting and pagination

Input validation and clean error handling

MongoDB with Mongoose

TypeScript and class-based OOP

Project Structure

src/
├── controllers/
├── services/
├── repositories/
├── models/
├── routes/
├── utils/
├── app.ts
└── server.ts

Tech Stack

Node.js, Express.js, TypeScript, Mongoose
